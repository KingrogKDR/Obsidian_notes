# 🧠 Hyprland Keybindings Cheat Sheet

_A quick reference for all your custom Hyprland shortcuts._

---

## 🧩 General Applications

| Keybind                   | Action                 |
| ------------------------- | ---------------------- |
| SUPER + B                 | Launch browser         |
| SUPER + O                 | Launch note-taking app |
| SUPER + Shift + O         | Launch office suite    |
| SUPER + Shift + E         | Alternate file manager |
| SUPER + Enter             | Terminal               |
| SUPER + Backspace         | Editor                 |
| SUPER + Shift + Backspace | Alternate editor       |

## 🔧 Resize Mode (Enter: `SUPER + R`)

| Keybind | Action           |
| ------- | ---------------- |
| ← / H   | Resize left      |
| → / L   | Resize right     |
| ↑ / K   | Resize up        |
| ↓ / J   | Resize down      |
| Esc     | Exit resize mode |

